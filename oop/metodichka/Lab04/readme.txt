Это пример выполнения лабораторной работы № 4.

Файл list.cpp содержит реализацию некоторых методов
класса List, остальные методы классов Item и List
уже скомпилированы и находятся в файле list_.obj.
Описание классов List и Item приведено в файле list.h
полностью. Файл main.cpp содержит реализацию главной
функции и класса Test - потомка класса List,
созданного для тестирования списка.

Чтобы запустить программу, необходимо создать проект,
руководствуясь указаниями к лабораторной работе № 4,
но кроме файлов main.cpp и list.cpp в состав проекта
необходимо также включить файл list_.obj.

Вместо файла main.cpp в проект можно включить файл
main_.cpp. Он идентичен по функциональности, но
иллюстрирует использование предопределенных потоков
стандартного ввода и вывода вместо функций printf и
scanf (с потоками ввода-вывода C++ предлагается
ознакомиться самостоятельно).

После успешной компиляции и запуска проекта можно
использовать предложенный готовый модуль списка (файлы
list.cpp, list_.obj и list.h) в составе созданного
проекта следующим образом:

1. При выполнении лабораторной работы № 4.
   Следует исключить из проекта скомпилированный модуль
   list_.obj и в файле list.cpp реализовать все остальные
   методы классов List и Item, руководствуясь заданием и
   указаниями к данной работе.

2. При выполнении лабораторных работ №№ 5-7.
   В полученный проект следует включить новый модуль
   subj.cpp, руководствуясь указаниями к работе № 5.
   В дальнейшем, сами работы выполняются в соответствии
   с заданием и указаниями к ним. Это не избавляет от
   необходимости выполнить работу № 4!
